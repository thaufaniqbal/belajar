international naming class, package, and variable.

penamaan class:
    di awali huruf besar dan kata selanjutnya di awali huruf besar
    contoh :
    1. HelloWorld.Java
    2. KamuJelek.Java
    3. KucingJantan.java
    4. AnjingGalak.Java

penamaan package :
    huruf kecil semua
    contoh :
    1. inikelashelloworld
    2. inifolderkamu

penamaan vaiable :
    huruf kecil di awal dan kata selanjutnya di awali huruf besar
    contoh :
    1. kamuJelek
    2. retriBalmond
    3. kucingJantan
    4. anjingGalak
